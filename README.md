# Node.js Practice Project

This is my project for practicing native Node.js.

## My Learning Files

- **`server.js`** – My Data API. Sends and receives data using Thunder Client.
- **`server2.js`** – My Website Server. Loads HTML pages into the browser.
- **`public/`** – Holds my website files (`index.html` and `about.html`).

## How to Run

Open your terminal and type:

```bash
node server.js   # To test your Data API
node server2.js  # To view your web pages
```
